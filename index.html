Welcome

Semi-classical signal analysis

Welcome to the main source of information for SCSA

This website is aimed to serve as a tool for understanding SCSA, it's applications and background theory. Using tools developed by the EMAN group (currently part of King Abdullah University of Technollogy) in platforms such as Matlab, Python and C++.

With this website we aim to achieve several things of which the principal one is to have it as foundation for a book having as central theme all about SCSA. Other objectives are having a free repository for people interested in this new signal analysis approach to tinker with and explore the possibilities of this method in ways that we may not have thought.

If you want to go ahead and read some of the theory behind all of this make sure to head to the Theory section. If you just want to tinker with the available tools then these are available under the software section. For further applications and research done make sure to have a look on the denoising and publications sections.
